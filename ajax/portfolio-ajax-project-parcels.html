Parcel Prioritization Tool

Project Info:

The Parcels application is a decision-support tool built to assist land trusts and non-profits in making strategic easement decisions. Through a secure web interface, authorized users can visualize and evaluate candidate properties for potential easements.

This tool was developed using the Umbraco .NET framework, with custom-built APIs that connect to a PostgreSQL database to serve vector tile data (MVTs) to the interactive web map. User permissions are managed within Umbraco, ensuring access is limited to approved users.

Project Details:

  • Client:Univeristy of Wyoming
  • Technologies:S3, HTML, CSS, C++, JavaScript, Postgres
