Mapping and Prioritizing Fences

Project Info:

This fence tool is a field-to-web system designed to support wildlife-friendly fencing efforts across the West. Using ArcGIS Field Maps, we developed a mobile data collection tool that allows partners to map fence lines, record fence types, wildlife interactions, and range improvements directly in the field. This information is stored in a secure, scalable database built for collaboration among landowners, agencies, and conservation groups.

Collected fence data is integrated into a custom web-based prioritization tool that identifies high-impact areas for modification based on wildlife movement patterns, habitat types, and land ownership.
