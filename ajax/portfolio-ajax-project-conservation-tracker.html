Conservation Tracker

Project Info:

The Conservation Tracker is a web-based tool designed to support landscape-scale conservation planning. For any selected area—like a migration corridor—it provides a high-level breakdown of land administration, conservation easements, private lands, and development, helping land managers identify critical areas and track progress toward conservation goals.

Built with a PostgreSQL backend, the app uses custom APIs to deliver data based on user-defined areas of interest. It visualizes hard-to-find datasets in an accessible format, while keeping sensitive details protected, making it a valuable tool for both planning and decision-making.

Project Details:

  • Client:University Of Wyoming
  • Technologies:S3, HTML, CSS, C++, JavaScript, Postgres
